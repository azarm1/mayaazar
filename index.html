Maze Runner – Java Graph Search Visualizer

Maze Runner is a Java-based application that converts mazes into graphs and finds optimal paths using Breadth-First Search. I followed SOLID principles and used GoF design patterns to structure the codebase. The project was developed using Maven for dependency management and tested thoroughly with JUnit.

Drone Rescue Mission – Java Map Navigation Tool

This project simulates a drone navigating a JSON-based island map to locate emergency zones and inlets. I implemented the logic using Java and applied GRASP design principles for clean structure and reusability. I used JUnit for test coverage and Log4J for debugging and tracing the drone’s path based on user input.
